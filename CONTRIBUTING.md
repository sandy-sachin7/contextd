# Contributing to contextd

Thanks for your interest in `contextd`! As this is currently a in the earliest of stages, the contribution process is simple.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on GitHub. Include as much detail as possible:
- Steps to reproduce
- Expected vs. actual behavior
- Logs or error messages

## Pull Requests

Pull requests are welcome!
1. Fork the repository.
2. Create a feature branch.
3. Make your changes (please include tests!).
4. Ensure `cargo test` passes.
5. Submit a PR.
